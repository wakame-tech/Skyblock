# Skyblock
## Features
- 島をGit管理
- Islands Manager
- Advancements DSL

## 🐳 start spigot server
```bash
docker-compose up -d
```

## 📚 attach logs
```bash
docker-compose exec spigot-server mc_log
```

## 🔥 build plugin
require `WorldEdit` in `plugins` Folder.

```bash
cd Skyblock
./gradlew build
```
